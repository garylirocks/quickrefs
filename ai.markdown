# AI

## Concepts

Machine learning (ML)

- Supervised
  - Labeled data
  - Best for prediction
- Unsupervised
  - Unlabeled data
  - Best for exploration
- Reinforcement
  - Reward-based learning
  - Best for decision making in dynamic environments

Other

- Assistant
  - Interact with humans
- Agent
  - Run in the background
  - Respond to events, or schedules
